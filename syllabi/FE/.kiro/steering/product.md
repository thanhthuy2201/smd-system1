# Product Overview

Shadcn Admin is a modern admin dashboard UI built with Shadcn UI components and Vite. It provides a reusable collection of dashboard interfaces for web applications.

## Key Features

- Responsive design with mobile support
- Accessibility-first approach
- Light/dark theme switching
- RTL (Right-to-Left) language support
- Global search command interface
- Built-in sidebar navigation
- 10+ pre-built pages (dashboard, tasks, users, settings, auth, etc.)
- Partial authentication integration with Clerk

## Target Use Case

This is a UI component library and reference implementation, not a starter template. It's designed to be adapted and integrated into existing projects that need admin dashboard interfaces.

## Authentication

Partial Clerk integration is included for demonstration. Auth flows include sign-in, sign-up, OTP verification, and forgot password pages.
